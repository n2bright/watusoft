# Watusoft

This repository contains the Watusoft landing page and supporting files for the Watusoft Solutions website.

## Contents
- index.html — main landing page
- files/ — supporting files such as the security checklist PDF

## Preview locally
Open index.html in a browser, or serve the folder with a simple static server:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000.
